About

This app's main purpose is to give users a solid estimated price for a given vehicle based on the car's make, model, and year. There will be more features to come in the future that will provide more information and give a bettere user experience. This app makes use of Python's Django framework to give a UI for a xgboost gradient boosting machine learning model. The data is mainly processed and stored using Pandas and a Postgresql database. As far as the front end, it makes use of Bootstrap to give a better viewing experience but it will continue to improve.
